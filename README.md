# Expense Tracker

A **Python-based Expense Tracker** with a polished console interface, data persistence, and visual analytics.

---

## Features

- Add, view, and filter expenses by category  
- Calculate total spending  
- Visualize spending trends with **pie charts** (category-wise) and **bar charts** (monthly) using Matplotlib  
- Persistent data storage in `expenses.txt`  
- Polished console interface with **colors** using Colorama  
- Fully safe fallback if Matplotlib or Colorama is not installed
